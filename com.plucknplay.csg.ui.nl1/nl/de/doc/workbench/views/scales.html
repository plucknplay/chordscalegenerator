Skalen Sicht

Die Skalen Sicht dient zum Verwalten all deiner Skalen. Ausgehend von dieser Sicht kannst du neue Skalen anlegen, bestehende Skalen editieren oder auch löschen. Außerdem ist es hier möglich, deine Skalen in Kategorien und Unterkategorien zu organisieren.

Doppelklick auf einer Skala öffnet diese in einem Skalen-Editor. Doppelklick auf einer Kategorie klappt die Kategorie wiederum auf oder zusammen.

Symbolleiste

Die Symbolleiste der Skalen-Sicht enthält die folgenden Funktionen:

Füge Skala hinzu

Öffnet einen leeren Skalen-Editor, in welchem du eine neue Skala definieren kannst. Nachdem du die neu definierte Skala abgespeichert hast (siehe Datei → Speichern), wird die Skala in die Kategorie der Skalenliste eingefügt, welche beim Erstellen der neuen Skala aktuell ausgewählt war. War zu diesem Zeitpunkt keine Kategorie ausgewählt, wird die neue Skala in die Hauptkategorie eingefügt.

Füge Kategorie hinzu

Fügt eine neue Unterkategorie zu der aktuell ausgewählten Kategorie hinzu. Du kannst den Namen der neuen Kategorie dabei direkt editieren.

Alles aufklappen

Klappt die gesamte Baumstruktur dieser Sicht auf, sodass sämtliche definierten Skalen sichtbar werden.

Alles zusammenklappen

Klappt die gesamte Baumstruktur dieser Sicht zusammen.

Verknüpfe mit Editor

Hier kannst du einstellen, ob die Skalen-Sicht mit dem aktuellen Editor verknüpft sein soll. Ist diese Option eingestellt, wird mit dem Wechsel zu einem aktuellen Skalen-Editor automatisch die dazugehörige Skala in der Skalen-Sicht selektiert.
Hinweis: Vor allem bei einer Vielzahl definierter Skalen ist eine Verknüpfung der Skalen-Sicht mit dem Editor sinnvoll, um den Überblick zu behalten.

Zeige Intervalle

Blendet die innere Intervallsicht ein oder aus. Die Intervallsicht zeigt zu einer aktuell ausgewählten Skala dessen definierte Intervalle an.

Vertikale Ausrichtung

Positioniert die innere Intervallsicht unterhalb der eigentlichen Skalen-Sicht.

Horizontale Ausrichtung

Positioniert die innere Intervallsicht rechts neben der eigentlichen Skalen-Sicht.

Automatische Ausrichtung

Positioniert die innere Intervallsicht abhängig von der Dimension der Skalen-Sicht. Ist die Sicht höher als breit, so wird die Sicht vertikal ausgerichtet. Ist die Sicht wiederum breiter als hoch, so wird die horizontale Ausrichtung verwandt.

Kontextmenü

Das Kontextmenü dieser Sicht enthält die folgenden Funktionen:

Neu → Skala
Neu → Kategorie

Ausschneiden

Schneidet die ausgewählten Elemente (Skalen und/oder Kategorien) aus und kopiert diese in die Zwischenablage.

Duplizieren

Dupliziert die ausgewählten Elemente, wobei die Namen der neuen Elemente um eine fortlaufende Nummer ergänzt werden.

Einfügen

Fügt die in der Zwischenablage befindlichen Elemente in die aktuelle ausgewählte Kategorie ein. Existieren in dieser Kategorie bereits gleichnamige Elemente, so werden die Namen der eingefügten Elemente um fortlaufende Nummern ergänzt.

Löschen

Löscht die ausgewählten Elemente. Löscht man eine Kategorie, so werden auch deren enthaltene Elemente gelöscht. Aber keine Angst, so schnell kann man nichts löschen - man muss vor jedem Löschvorgang, diesen über einen entsprechenden Dialog bestätigen.

Umbenennen

Ermöglicht das direkte Umbenennen eines ausgewählten Elements (Skala oder Kategorie).

Importieren...

Importiert die Elemente einer ausgewählten XML-Datei und fügt diese in die aktuelle ausgewählte Kategorie ein.

Exportieren...

Exportiert die ausgewählten Elemente in eine angegebene XML-Datei.

Kopiere Skalenblöcke in die Zwischenablage

Kopiert die Bilddaten aller Skalenblöcke einer selektierten Skala in die Zwischenablage. Welche Skalenblöcke einer selektierten Skala exportiert werden, hängt dabei von dem festgelegten Skalenblockmodus ab (siehe Benutzervorgaben: Blöcke). Die Bilddaten können dann beispielsweise in Office-Anwendungen eingefügt werden. Welche Bilddaten hierbei in die Zwischenablage kopiert werden, kannst du in den Benutzervorgaben zum Export spezifizieren.
Beachte: Bei der Größe der exportierten Bilddaten werden die in den Benutzervorgaben. definierten festen Größen verwendet. Die aktuelle Größe von Sichten kann hier nicht verwendet werden, da eine Sicht in mehreren Perspektiven mit unterschiedlicher Größe geöffnet sein kann.
