Akkorde Sicht

Die Akkorde Sicht dient zum Verwalten all deiner Akkorde. Ausgehend von dieser Sicht kannst du neue Akkorde anlegen, bestehende Akkorde editieren oder auch löschen. Außerdem ist es hier möglich, deine Akkorde in Kategorien und Unterkategorien zu organisieren.

Doppelklick auf einem Akkord öffnet diesen in einem Akkord-Editor. Doppelklick auf einer Kategorie klappt die Kategorie wiederum auf oder zusammen.

Symbolleiste

Die Symbolleiste der Akkorde-Sicht enthält die folgenden Funktionen:

Füge Akkord hinzu

Öffnet einen leeren Akkord-Editor, in welchem du einen neuen Akkord definieren kannst. Nachdem du den neu definierten Akkord abgespeichert hast (siehe Datei → Speichern), wird der Akkord in die Kategorie der Akkordliste eingefügt, welche beim Erstellen des neuen Akkords aktuell ausgewählt war. War zu diesem Zeitpunkt keine Kategorie ausgewählt, wird der neue Akkord in die Hauptkategorie eingefügt.

Füge Kategorie hinzu

Fügt eine neue Unterkategorie zu der aktuell ausgewählten Kategorie hinzu. Du kannst den Namen der neuen Kategorie dabei direkt editieren.

Alles aufklappen

Klappt die gesamte Baumstruktur dieser Sicht auf, sodass sämtliche definierten Akkorde sichtbar werden.

Alles zusammenklappen

Klappt die gesamte Baumstruktur dieser Sicht zusammen.

Verknüpfe mit Editor

Hier kannst du einstellen, ob die Akkorde-Sicht mit dem aktuellen Editor verknüpft sein soll. Ist diese Option eingestellt, wird mit dem Wechsel zu einem aktuellen Akkord-Editor automatisch der dazugehörige Akkord in der Akkorde-Sicht selektiert. Hinweis: Vor allem bei einer Vielzahl definierter Akkorde ist eine Verknüpfung der Akkorde-Sicht mit dem Editor sinnvoll, um den Überblick zu behalten.

Zeige Intervalle

Blendet die innere Intervallsicht ein oder aus. Die Intervallsicht zeigt zu einem aktuell ausgewählten Akkord dessen definierte Intervalle an.

Menü

Das Menü der Akkorde-Sicht enthält die folgenden Funktionen:

Bereinigen...

Öffnet den Wizard zum Bereinigen der Akkordliste. (siehe auch: Menü → Extras → Bereinige Akkorde...).

Füge Kategorie hinzu

Vertikale Ausrichtung

Positioniert die innere Intervallsicht unterhalb der eigentlichen Akkorde-Sicht.

Horizontale Ausrichtung

Positioniert die innere Intervallsicht rechts neben der eigentlichen Akkorde-Sicht.

Automatische Ausrichtung

Positioniert die innere Intervallsicht abhängig von der Dimension der Akkorde-Sicht. Ist die Sicht höher als breit, so wird die Sicht vertikal ausgerichtet. Ist die Sicht wiederum breiter als hoch, so wird die horizontale Ausrichtung verwandt.

Kontextmenü

Das Kontextmenü dieser Sicht enthält die folgenden Funktionen:

Neu → Akkord
Neu → Kategorie

Ausschneiden

Schneidet die ausgewählten Elemente (Akkord und/oder Kategorien) aus und kopiert diese in die Zwischenablage.

Duplizieren

Dupliziert die ausgewählten Elemente, wobei die Namen der neuen Elemente um eine fortlaufende Nummer ergänzt werden.

Einfügen

Fügt die in der Zwischenablage befindlichen Elemente in die aktuelle ausgewählte Kategorie ein. Existieren in dieser Kategorie bereits gleichnamige Elemente, so werden die Namen der eingefügten Elemente um fortlaufende Nummern ergänzt.

Löschen

Löscht die ausgewählten Elemente. Löscht man eine Kategorie, so werden auch deren enthaltene Elemente gelöscht. Aber keine Angst, so schnell kann man nichts löschen - man muss vor jedem Löschvorgang, diesen über einen entsprechenden Dialog bestätigen.

Umbenennen

Ermöglicht das direkte Umbenennen eines ausgewählten Elements (Akkord oder Kategorie).

Importieren...

Importiert die Elemente einer ausgewählten XML-Datei und fügt diese in die aktuelle ausgewählte Kategorie ein.

Exportieren...

Exportiert die ausgewählten Elemente in eine angegebene XML-Datei.
