Instrumente Sicht

Die Instrumente Sicht dient zum Verwalten all deiner Instrumente. Ausgehend von dieser Sicht kannst du neue Instrumente anlegen, bestehende Instrumente editieren oder auch löschen. Außerdem ist es hier möglich, deine Instrumente in Kategorien und Unterkategorien zu organisieren.

Doppelklick auf einem Instrument öffnet dieses in einem Instrumenten-Editor. Doppelklick auf einer Kategorie klappt die Kategorie wiederum auf oder zusammen.

Symbolleiste

Die Symbolleiste der Instrumente-Sicht enthält die folgenden Funktionen:

Füge Instrument hinzu

Öffnet einen leeren Instrumenten-Editor, in welchem du ein neues Instrument definieren kannst. Nachdem du das neu definierte Instrument abgespeichert hast (siehe Datei → Speichern), wird das Instrument in die Kategorie der Instrumenteliste eingefügt, welche beim Erstellen des neuen Instruments aktuell ausgewählt war. War zu diesem Zeitpunkt keine Kategorie ausgewählt, wird das neue Instrument in die Hauptkategorie eingefügt.

Füge Kategorie hinzu

Fügt eine neue Unterkategorie zu der aktuell ausgewählten Kategorie hinzu. Du kannst den Namen der neuen Kategorie dabei direkt editieren.

Alles aufklappen

Klappt die gesamte Baumstruktur dieser Sicht auf, sodass sämtliche definierten Instrumente sichtbar werden.

Alles zusammenklappen

Klappt die gesamte Baumstruktur dieser Sicht zusammen.

Verknüpfe mit Editor

Hier kannst du einstellen, ob die Instrumente-Sicht mit dem aktuellen Editor verknüpft sein soll. Ist diese Option eingestellt, wird mit dem Wechsel zu einem aktuellen Instrumenten-Editor automatisch das dazugehörige Instrument in der Instrumente-Sicht selektiert.
Hinweis: Vor allem bei einer Vielzahl definierter Instrumente ist eine Verknüpfung der Instrumente-Sicht mit dem Editor sinnvoll, um den Überblick zu behalten.

Kontextmenü

Das Kontextmenü dieser Sicht enthält die folgenden Funktionen:

Neu → Instrument
Neu → Kategorie

Ausschneiden

Schneidet die ausgewählten Elemente (Instrumente und/oder Kategorien) aus und kopiert diese in die Zwischenablage.

Kopieren

Kopiert die ausgewählten Elemente in die Zwischenablage.

Einfügen

Fügt die in der Zwischenablage befindlichen Elemente in die aktuelle ausgewählte Kategorie ein. Existieren in dieser Kategorie bereits gleichnamige Elemente, so werden die Namen der eingefügten Elemente um fortlaufende Nummern ergänzt.

Löschen

Löscht die ausgewählten Elemente. Löscht man eine Kategorie, so werden auch deren enthaltene Elemente gelöscht. Aber keine Angst, so schnell kann man nichts löschen - man muss vor jedem Löschvorgang diesen über einen entsprechenden Dialog bestätigen.

Umbenennen

Ermöglicht das direkte Umbenennen eines ausgewählten Elements (Instrument oder Kategorie).

Importieren...

Importiert die Elemente einer ausgewählten XML-Datei und fügt diese in die aktuelle ausgewählte Kategorie ein.

Exportieren...

Exportiert die ausgewählten Elemente in eine angegebene XML-Datei.

Setze aktiv

Setzt das ausgewählte Instrument aktiv. Das aktive Instrument gibt an, auf welchem Instrument alle folgenden Berechnungen (Grifftabellen generieren, Akkorde finden, ...) aktuell durchgeführt werden sollen.
