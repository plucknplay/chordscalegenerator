Skalensuche Sicht

Die Sicht Skalensuche dient zum Spezifizieren und Starten von Berechnungen zum Finden von passenden Skalen zu einer Akkord- bzw. Notenfolge. Die Ergebnisse der Berechnung werden in der Skalen-Ergebnisse-Sicht angezeigt.

Drop Support

Die Sicht Skalensuche unterstützt auch Drag&Drop - also das Ziehen von Elementen aus einer Sicht in eine andere. In diesem Fall ist es möglich, dass Akkorde aus der Akkorde-Sicht oder der Akkord-Ergebnisse-Sicht in diese Sicht gezogen werden können. Läßt man einen Akkord in dieser Sicht fallen (droppen), so wird der entsprechende Akkord automatisch hinzugefügt.

Symbolleiste

Die Symbolleiste der Skalensuche-Sicht enthält lediglich die folgende Funktion:

Lösche Eingabe

Löscht die aktuelle Eingabe (Akkorde und Noten) dieser Sicht.

Menü

Das Menü der Skalensuche-Sicht enthält die folgenden Funktionen:

Vertikale Ausrichtung

Positioniert den Bereich zur Definition der Notenfolge unterhalb von dem Bereich zur Definition der Akkordfolge.

Horizontale Ausrichtung

Positioniert den Bereich zur Definition der Notenfolge neben dem Bereich zur Definition der Akkordfolge.

Automatische Ausrichtung

Positioniert den Bereich zur Definition der Notenfolge abhängig von der Dimension der Skalensuche-Sicht.

Lösche Eingabe

Benutzervorgaben...

Öffnet den Dialog zum Definieren der Benutzervorgaben. Die Seite zum Definieren der Einstellmöglichkeiten für diese Sicht ist dabei schon vorselektiert.

Verwandte Themen

Arbeitsfläche → Sichten → Akkorde
Arbeitsfläche → Sichten → Akkord-Ergebnisse
Arbeitsfläche → Sichten → Skalen-Ergebnisse

Verwandte Aufgaben

Aufgaben → Skalen finden
