Menü: Datei

Das Menü Datei des Hauptmenüs enthält die folgenden Funktionen:

Neu → Instrument

Öffnet einen leeren Instrumenten-Editor, in welchem du ein neues Instrument definieren kannst. Nachdem du das neu definierte Instrument abgespeichert hast (siehe Datei → Speichern), wird das Instrument in die Hauptkategorie der Instrumenteliste eingefügt.

Neu → Akkord

Öffnet einen leeren Akkord-Editor, in welchem du einen neuen Akkord definieren kannst. Nachdem du den neu definierten Akkord abgespeichert hast (siehe Datei → Speichern), wird der Akkord in die Hauptkategorie der Akkordliste eingefügt.

Neu → Skala

Öffnet einen leeren Skalen-Editor, in welchem du eine neue Skala definieren kannst. Nachdem du die neu definierte Skala abgespeichert hast (siehe Datei → Speichern) , wird die Skale in die Hauptkategorie der Skalenliste eingefügt.

Speichern

Speichert das in dem aktiven Editor befindlichen Element (Instrument, Akkord oder Skala) ab.

Alle speichern

Speichert alle aktuell in einem Editor befindlichen Elemente ab.

Benutzervorgaben...

Öffnet den Dialog zum Definieren der Benutzervorgaben. Die zuletzt ausgewählte Seite wird dabei vorselektiert.

Beenden

Beendet die Anwendung.
