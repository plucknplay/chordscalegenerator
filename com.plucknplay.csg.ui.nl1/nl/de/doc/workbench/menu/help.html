Menü: Hilfe

Das Menü Hilfe des Hauptmenüs enthält die folgenden Funktionen:

Willkommen

Öffnet die Willkommensseiten des Chord Scale Generators. In den Willkommensseiten wird ein erster Überblick sowie eine Rundtour zum Programm gegeben.

Inhalt der Hilfetexte

Öffnet die Hilfetexte in einem Browser. Ob ein interner oder externer Browser verwendet werden soll, kannst du in den Benutzervorgaben zur Hilfe spezifizieren.

Suchen

Öffnet die Hilfe-Sicht zur Eingabe eines Suchausdrucks.

Dynamische Hilfe

Zeigt nützliche Hilfetexte für ein ausgewähltes Programmelement (Sicht, Editor oder Benutzervorgaben-Seite) an. Hierbei wird sowohl die zum Programmelement gehörende Hilfeseite aufgeführt sowie weitere möglicherweise interessante Hilfetexte.

Video-Tutorials

This menu entry takes you directly to the tutorial page of the project's website.

Support

This menu entry takes you directly to the support page of the project's website. The support page lists the frequently asked questions (FAQ).

Was ist neu?

This menu entry takes you directly to the latest release notes on the project's website.

Suche nach Updates

Überprüft, ob neue Updates verfügbar sind.

Info

Öffnet den Info-Dialog dieser Anwendung.
