Update-Funktionalität

Im letzten Kapitel dieser kleinen Rundtour soll nun noch abschließend auf die Update-Funktionalität des Chord Scale Generators eingegangen werden. Der Chord Scale Generator verfügt über eine Update-Funktion, womit du dein Programm auf den neuesten Stand bringen kannst. Du musst also nicht das Programm jedes mal neu installieren, wenn eine neue Version verfügbar ist.

Updates stellen kleine Verbesserungen des Programms dar. Dabei werden entweder mit einem Update neuen Funktionen bereitgestellt oder kleine Fehler behoben, die sich zuvor in das Programm eingeschlichen hatten. Es lohnt sich demnach regelmäßig zu überprüfen, ob neue Updates verfügbar sind. Aber keine Angst, bei einem Update werden die aktuellen Einstellungen oder Daten (Instrumente, Akkorde oder Skalen) deiner Anwendung natürlich nicht verändert.

Die Suche nach neu zu installierenden Updates kannst du unter Hilfe → Suche nach Updates starten. Damit überprüft werden kann, ob eine neue Version des Chord Scale Generator verfügbar ist, muss für den Update-Prozess eine aktive Internet-Verbindung vorliegen. Wird eine neue Version gefunden, erscheint ein entsprechender Dialog, welcher dir die neue Version anzeigt. Nachdem du nun noch den Lizenzangaben zugestimmt hast, wird die neue Version heruntergeladen und installiert. Abschließend solltest du den Chord Scale Generator nun nur noch neu starten, damit die Veränderungen wirksam werden können.
Achtung: Das Herunterladen der neuen Version kann abhängig von der Geschwindigkeit deiner Internet-Verbindung einige Zeit in Anspruch nehmen. Du kannst den Update-Prozess aber auch im Hintergrund laufen lassen, sodass du den Chord Scale Generator währenddessen benutzen kannst.

Es ist auch möglich, die Suche nach Updates automatisch durchführen zu lassen. Hierfür kannst du in den Benutzervorgaben zum Update genau einstellen, wann die Suche nach Updates automatisch ausgeführt werden soll.

Verwandte Themen

Arbeitsfläche → Menü → Hilfe
Arbeitsfläche → Benutzervorgaben → Update
