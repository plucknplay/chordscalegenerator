Scale Editor

In the Scale Editor you can define new scales and edit existing ones.

Options

When you define a scale, you have the following options:

Name

Here you can define the name of your scale.

aka

Here you can define additional aka names of the scale. (Aka stands for "also known as".) This option takes into account that most of the scales have several different synonyms. With the help of the "aka names" you do not have to define a separate scale for each of the synonyms. These additionally defined names can also help if you want to find a scale. The global action Extras → Open Scale... considers all defined names (including the aka names) of your scales.

Intervals

Here you can specify the intervals which define the scale. Each row represents an interval. In the rows itself all possible names of an individual interval are listed. The most common names are highlighted with a grey background color. In order to add an interval to the scale you just have to check the name of the interval you want.

Comment

In the comment section you can add any kind of information to your scale. The comments have no influence on any calculations.
