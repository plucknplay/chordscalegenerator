Help View

The Help View provides you assistance inside the workbench. The view consists of several pages. Each page presents help in a different fashion. Hyperlinks at the bottom of the view allow switching among pages. Clicking any topic will display its contents.

Pages

The Help View contains the following pages:

Related Topics

The Related Topics Page shows descriptions and help topics related to the currently selected element (editor or view). The About Section shows context help specific to your current context, and the Dynamic Help Section shows some search results that may be related. This page tracks the changes in the workbench and continuously updates displayed information.

All Topics

The All Topics Page shows the table of contents. It is a hierarchy of all help topics.

Search

The Search Page allows browsing local topics and other documents given a search query. Links to the search results are displayed along with a summary of the topic contents. The search scope controls a subset of documentation being searched.

Bookmarks

The Bookmarks Page shows topics marked as personal bookmarks.

Toolbar

The toolbar of the Help View contains the following actions. The available actions depend on the currently displayed page.

Back

Returns to the last page or topic you have viewed.

Forward

Once you have pressed the Back button, the Forward button moves you ahead through pages and topics you have already viewed prior to the current display.

Collapse All

Collapses the expanded tree structure of all topics on the page.

Show All Topics

When the filtering of workbench elements is enabled, this action enables displaying topics for disabled elements.

Show in external window

Display the currently selected topic in an external window.

Show in All Topics

Shows the currently displayed topic in the table of contents hierarchy.

Bookmark document

Bookmarks the currently displayed document.

Show Result Categories

Switches between sorting search results by relevance and by logical containers (for example books).

Show Result Descriptions

Displays a short description of each search hit, if available.
