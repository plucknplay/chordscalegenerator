Keyboard View

The Keyboard View shows chords and scales on the keyboard of a grand piano. You can choose between a notes and a interval mode.

A currently selected scale block (if available) can be highlighted with blue keys.

In the Keyoard View the tones of a currently played chord or scale block are highlighted. Thus, you can see exactly which tone is currently played and for how long.

Besides, it is possible to play freely on the keyboard. A simple mouse click plays the selected key. You can define in the sound preferences how long a played tone shall sound.

Toolbar

The toolbar of the Keyboard View contains the following actions:

Show Notes

Switches to the notes mode of this view. The notes of the currently selected chord or scale block will be shown.

Show Intervals

Switches to the interval mode of this view. The intervals of the currently selected chord or scale block will be shown.

Clear Input

Clears the currently shown input (chord or scale) from this view.

Menu

The menu of the Keyboard View contains the following actions:

Show Notes
Show Intervals
Clear Input

Preferences...

Opens the preferences dialog. The page with the preferences for this view is already pre-selected.

Pop-up Menu

The pop-up menu of this view contains the following actions:

Export as Image File...

Exports the current visual representation of the keyboard as an image file. You can specifiy the file name and type (.bmp, .jpg or .png) in a corresponding dialog box. The size of the export image can be defined in the export preferences.

Copy to Clipboard

Copies the current visual representation of the keyboard into the clipboard. The stored image data in the clipboard may be pasted, for instance, into Office applications later on. The size of the image can be defined in the export preferences as well.
