Instruments View

In the Instruments View you can manage all your instruments. From here, you can create new instruments, edit existing instruments or even delete existing instruments. Besides, it is possible to organize your instruments in categories and subcategories.

Double-click on an instrument opens the instrument in the Instrument Editor. Double-clicking on a category expands or collapses the category.

Toolbar

The toolbar of the Instruments View contains the following actions:

Add Instrument

Opens a blank Instrument Editor where you can define a new instrument. After you have saved the new instrument (see File → Save), it will be inserted into the category which was selected during the creation of the new instrument. If there was no category selected the new instrument will be added into the main category.

Add Category

Adds a new sub-category into the currently selected category. The name of the category can be edited directly.

Expand All

Expands the whole tree structure of this view. All defined instruments become visible.

Collapse All

Collapses the whole tree structure of this view.

Link with Editor

Toggles whether or not the Instruments View selection is linked to the active editor. When this option is selected, changing to an active Instrument Editor will automatically update the Instruments View selection to the instrument being edited.
Note: Especially with a multitude of defined instruments it may be useful to link the Instruments View with the editor for the sake of clarity.

Menu

The menu of the Instruments View contains the following actions:

Add Instrument
Add Category
Expand All
Collapse All

Pop-up Menu

The pop-up menu of this view contains the following actions:

New → Instrument
New → Category

Cut

Cuts out the selected elements (instruments and/or categories) and copies them into the clipboard.

Copy

Copies the selected elements into the clipboard.

Paste

Pastes the elements of the clipboard into the currently selected category. If the selected category contains elements with the same name a consecutive number will be added to the element name.

Remove

Removes the selected elements. If you remove a category all contained elements of the category will be removed as well. Each removal has to be confirmed through a dialog box.

Rename

Enables the direct renaming of a selected element (instrument or category).

Import...

Imports the elements of a chosen XML-file. The elements will be added into the currently selected category.

Export...

Exports the selected elements into a specified XML-file.

Set Active

Sets the selected instrument active. All calculations (Generate Chords, Find Chords, ...) will be performed on the active instrument.
