Notes View

The Notes View shows chords and scales in musical notation. It is possible to change between various clefs.

This view not only shows chords and scale blocks - here you can also switch to a edit/search mode. In the edit/search mode you can arrange own notations and start a reverse calculation to find chords and scales. In order to arrange your notation just double click to add new notes or remove existing notes.

Toolbar

The toolbar of the Notes View contains the following actions:

Edit/Search Mode

Here you can switch to the edit/search mode. In the edit/search mode you can arrange own notations and start a reverse calculation to find chords and scales.
Note: As long as the view is in the edit/search mode, it is blocked. That means that newly selected chords or scale blocks will not be shown in the view.

Find Chord

This action starts the calculation to determine the corresponding chords of a notation you have arranged in the edit/search mode. This action is only available if it is actually possible that the notation corresponds to a chord. Therefore, the notation must contain at least two notes.

Find Scale

This action starts the calculation to determine the corresponding scales of a notation you have arranged in the edit/search mode. This action is only available if the notation contains at least two notes.

Accidental Lock Mode

Here you can activate or deactivate the Accidental Lock Mode. If the Accidental Lock Mode is activated new notes will be added with the corresponding sign. Which accidental will be added to new notes depends on the globally set notes mode.

Clear Input

Clears the currently shown input (chord or scale) from this view.

Menu

The menu of the Notes View contains the following actions:

Clef Menu

Here you can choose which clef should be used in this view. Depending on the clef (you choose), the position of the notes will change.
Note: It is possible that existing notes will move outside of the visible area of this view when you change the clef. If this happens the chosen clef is probably not the right one for the currently active instrument.

Preferences...

Opens the preferences dialog. The page with the preferences for this view is already pre-selected.

Pop-up Menu

The pop-up menu of this view contains the following actions:

Export as Image File...

Exports the current representation of the notation as an image file. You can specifiy the file name and type (.bmp, .jpg or .png) in a corresponding dialog box. The size of the export image can be defined in the export preferences.

Copy to Clipboard

Copies the current representation of the notation into the clipboard. The stored image data in the clipboard may be pasted, for instance, into Office applications later on. The size of the image can be defined in the export preferences as well.
