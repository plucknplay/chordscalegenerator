Scale Results View

The Scale Results View shows the results of the calculation to find scales for a corresponding fingering or musical notation.

The scale result set will be presented in a tabular form. The (found) scales can be arranged in columns. The columns itself can be resized. Besides, unnecessary columns can be set invisible (see preferences of this view).

Since the result set may be very large, this view is divided into pages for the sake of clarity. If many pages exist, the current page will be shown in the toolbar on the left side of the actions to turn the pages.
Note: The arrangement in columns is not restricted to the current page - but takes into consideration the whole result set.

Table columns

Scale

This columns shows the complete name of the scale.

Intervals

This column lists the individual intervals of the scale.

Notes

This column lists the individual notes of the scale.

Reference Intervals

This column lists the intervals which correspond to the notes that were used for the calculation of the present scale result set. These listed intervals represent in most cases just a subset of the original intervals of the scale.

Reference Notes

This column lists the notes that were used for the calculation of the present scale result set. These listed notes represent in most cases just a subset of the original notes of the scale.

FT (Root Note)

This column indicates whether or not the reference notes include the root note of the scale.

Coverage

This column indicates which percentage of the notes of the scale is covered by the reference notes (all the notes that were used for the calculation)

Toolbar

The toolbar of the Scale Results View contains the following actions:

First Page

Jumps to the first result page of this view.

Previous Page

Jumps to the previous result page.

Next Page

Jumps to the next result page.

Last Page

Jumps to the last result page of this view.

Menu

The menu of the Scale Results View contains the following actions:

Preferences...

Opens the preferences dialog. The page with the preferences for this view is already pre-selected.

Pop-up menu

The pop-up menu of this view contains the following actions:

Open in Editor

Opens the selected scale in a corresponding Scale Editor.

Copy Scale Blocks to Clipboard

Copies the image data of all scale blocks of a selected scale (of this view) into the clipboard. Which scale blocks of a selected scale will be exported depends on which scale block mode you have selected (see Blocks Preferences). The stored image data may be pasted, for instance, into Office applications later on. You can define in the export preferences which image data shall be copied.
Note: The image data will be copied with the fixed sizes as defined in the export preferences. The current sizes of the views can not be used since a view can be opened in several perspectives with different sizes.

Play Scale

Plays the selected scale. You can define in the sound preferences, how a played scale should sound.
