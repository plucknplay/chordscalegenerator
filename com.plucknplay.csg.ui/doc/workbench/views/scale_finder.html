Scale Finder View

In the Scale Finder View you can specify and start calculations to find scales that match a chord or note sequence. The results of the calculation will be displayed in the Scale Results View.

Drop Support

The Scale Finder View supports Drag&Drop - the possibility to drag elements from one view to another. In this case it is possible to drag chords from the Chords View and the Chord Results View into this view. If you drop a chord into this view the chord will be automatically added.

Toolbar

The toolbar of the Scale Finder View contains only the following actions:

Clear Input

Clears the currently set input (chords and notes) of this view.

Menu

The menu of the Scale Finder View contains the following actions:

Vertical View Orientation

Positions the notes section underneath the chords section.

Horizontal View Orientation

Positions the notes section next to the chords section.

Automatic View Orientation

Positions the notes section depending on the dimension of this view.

Clear Input

Preferences...

Opens the preferences dialog. The page with the preferences for this view is already pre-selected.
