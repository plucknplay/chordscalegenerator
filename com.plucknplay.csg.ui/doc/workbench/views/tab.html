Tab View

The Tab View shows chords and scales in the form of a tablature. This representation is, just like the Box represenation, a very compact and simple one.

This view not only shows chords and scale blocks - here you can also switch into a edit/search mode. In the edit/search mode you can arrange own fingering patterns and perform a reverse calculation to find chords and scales. In order to arrange your fingering patterns just double click to select, and afterwards edit, a finger position.

Toolbar

The toolbar of the Tab View contains the following actions:

Edit/Search Mode

Here you can switch to the edit/search mode. In the edit/search mode you can arrange own fingering patterns and start a reverse calculation to find chords and scales.
Note: As long as the view is in the edit/search mode, it is blocked and newly selected chords or scale blocks will not be shown.

Find Chord

This action starts the calculation to determine the corresponding chords of a fingering pattern you have arranged before. This action is only available if it is actually possible that the fingering corresponds to a chord. Thus, the fingering pattern must contain at least two notes and only one finger position on each string can be defined.

Find Scale

This action starts the calculation to determine the corresponding scales of a fingering pattern you have arranged in the edit/search mode. This action is only available if the fingering pattern contains at least two notes. When you want to find scales it is, of course, possible to define more than one finger position on each string.

Remove Column

Removes a column from the tablature. The minumum number of columns is limited to one.

Add Column

Adds an additional column to the tablature. The maximum number of columns is limited to six.

Clear Input

Clears the currently shown input (chord or scale) from this view.

Menu

The menu of the Tab View contains the following actions:

Edit/Search Mode
Find Chord
Find Scale
Remove Column
Add Column
Clear Input

Preferences...

Opens the preferences dialog. The page with the preferences for this view is already pre-selected.

Pop-up Menu

The pop-up menu of this view contains the following actions:

Export as Image File...

Exports the current representation of the tablature as an image file. You can specifiy the file name and type (.bmp, .jpg or .png) in a corresponding dialog box. The size of the export image can be defined in the export preferences.

Copy to Clipboard

Copies the current representation of the tablature into the clipboard. The stored image data in the clipboard may be pasted, for instance, into Office applications later on. The size of the image can be defined in the export preferences as well.
