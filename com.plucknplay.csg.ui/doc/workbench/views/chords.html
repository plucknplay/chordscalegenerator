Chords View

In the Chords View you can manage all the chords. You can create new chords, edit existing chords or even delete existing chords. Besides, it is possible to organize the chords in categories and sub-categories.

Double-click on a chord opens the chord in the Chord Editor. Double-click on a category expands or collapses the category.

Toolbar

The toolbar of the Chords View contains the following actions:

Add Chord

Opens a blank Chord Editor where you can define a new chord. After you have saved the new chord (see File → Save), it will be inserted into the category which was selected during the creation of the new chord. If you have not selected a category the new chord will be added into the main category.

Add Category

Adds a new sub-category into the currently selected category. The name of the category can be edited directly.

Expand All

Expands the whole tree structure of this view. All defined chords become visible.

Collapse All

Collapses the whole tree structure of this view.

Link with Editor

Toggles whether or not the Chords View selection is linked to the active editor. When this option is selected, changing to an active Chord Editor will automatically update the chord being edited in the Chords View selection. Note: Especially with a multitude of defined chords it may be useful to link the Chords View with the editor for the sake of clarity.

Show Intervals

Makes the inner Interval View visible or invisible. The Interval View shows the defined intervals of a currently selected chord.

Menu

The menu of the Chords View contains the following actions:

Clean Up...

Opens the wizard for the chord clean-up. (see: Menu → Extras → Clean Up Chords...).

Add Chord
Add Category
Expand All
Collapse All
Link with Editor
Show Intervals

Vertical View Orientation

Sets the inner Interval View in a vertical alignment within the Chords View.

Horizontal View Orientation

Sets the inner Interval View in a horizontal alignment within the Chords View.

Automatic View Orientation

Sets the alignment of the inner Interval View depending on the dimension of the Chords View. If the length of the view is higher than its width the inner Interval View is set in a vertical alignment within the Chords View. In the reverse case, the horizontal alignment will be used.

Pop-up Menu

The pop-up menu of this view contains the following actions:

New → Chord
New → Category

Cut

Cuts out the selecteded elements (chords and/or categories) and copies them into the clipboard.

Duplicate

Duplicates the selected elements. New elements will be extended with a consecutive numbering.

Paste

Pastes the elements of the clipboard into the currently selected category. If the selected category contains elements with the same name a consecutive number will be added to the element name.

Remove

Removes the selected elements. If you remove a category all contained elements of the category will be removed as well. Each removal has to be confirmed through a dialog box.

Rename

Enables the direct renaming of a selected element (chord or category).

Import...

Imports the elements of a chosen XML-file. The elements will be added into the currently selected category.

Export...

Exports the selected elements into a specified XML-file.
