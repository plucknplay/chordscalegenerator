Preferences: Calculations

You can set the following preferences for the calculations:

Max. Number of Chord Results

Here you can define the maximum number of chord results for the chord generation. This limitation of the result set is necessary since unfavourable conditions (instruments with many strings, big fret ranges, etc.) may cause huge result sets with up to several million results. In such cases the calculation would take relatively long and the huge result sets would be almost unmanageable. So it is better to predefine the maximum number of results you are willing to revise.

Prefer Barre (chords)

Here you can specify whether you prefer barre chords or not. Some chords can be played with or without barre. It depends on you which fingering you prefer. If you have no problems with fretting barre chords you should probably select this setting, since barre chords need less fingers.

Find Chord

Here you can specifiy whether chords with a left out root, third and/or fifth should be included in the Find Chord calculation. Besides, you can specify how many intervals may be excluded at the most.
