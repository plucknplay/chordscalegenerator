Preferences: Warnings

It may happen that warning dialogs appear before or after perfoming certain actions which could cause unexpected side effects. You have the possibitiy to suppress the particular warning in the future.
Note: In the warning preferences you can also activate or deactivate warnings. Therefore, it is possible to reactivate suppressed warnings.

The warnings which may appear are the following:

Hide warning: Changing the active instrument ...

If you change the active instrument or modify the currently active instrument the Chord Results View will be cleared. This is necessary since the calculated chords belong to the active instrument. Another instrument usually has a completely different result set.

Hide warning: Changing the 'prefer barre' option ...

Changing the 'Prefer Barre (Chords)' settings in the calculations preferences will lead to a recalculation of the levels of the chords in the Chord Results View. Thus, it may happen that the recalculated levels do not match with the maximum levels you have determined in the Chord Generation View before starting the initial calculation.

Hide warning: The result set exceeds ...

If the size of the result set for a chord generation was bigger than the defined limit in the calculations preferences the result set will be reduced to its maximum allowed size.

Hide warning: No results...

If you are still using the test-version of the Chord Scale Generator it might (frequently) happen that the calculations "Find Chord" and "Find Scale" don't find any results. This behaviour can usually be explained with the reduced database of the test-version.
