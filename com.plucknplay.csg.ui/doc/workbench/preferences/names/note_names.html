Preferences: Note Names

Here, you can set general preferences for note names. The problem is, that some note names change depending on where you are living. In some language areas, for example, they use an H, whereas other languages areas use a B instead. For the lowered note there even exist three possible names with Hb, B and Bb.

In this preferences page you can set the note names which shall be used in the program.

Note: The settings on this page influence the representation of several views.
