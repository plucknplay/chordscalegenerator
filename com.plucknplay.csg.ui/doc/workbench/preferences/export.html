Preferences: Export

You can set the following preferences for the export:

File Extension

Here you can specifiy which file format shall be used for the image export into the clipboard. The chosen file extension will also represent the default selection of the export file dialog.

Export Size

Here you can specify the size of the exported image data of each individual graphical view. It is possible to choose between a fixed height (in pixels) or the current size of the view. The specified size will be used for any image file export as well as for copying image data into the clipboard.

Results Views - Clipboard Export

Here you can set preferences for the export of image data. You can specify which chords in the Chord Results View or which scale blocks of a selected scale in the Scale Results View shall be exported. Possible imaga data are the box, tab and notes representations (of the individual chords and scale blocks). Here you can define which representations you want to export and in which order.

Copy Graphics to Clipboard in Reverse Order

Here you can specify whether or not graphics that are copied to the clipboard shall be inserted in reverse order. This option can adjust the export to word processing programs that paste graphics from the clipboard in reverse order.
