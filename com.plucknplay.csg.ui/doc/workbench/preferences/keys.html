Preferences: Keys

Here you can define key bindings for all commands of the Chord Scale Generator. Key bindings allow you to additinally execute a command by its defined key sequence. There are already pre-defined key bindings for the most important commands. Of course, it's also possible to remove or modify these bindings.

In case you have assigned the same key sequence to several commands you will be notified about this conflict in the corresponding section at the bottom right corner of this dialog. Besides, you can restore the default values at any time.
