Preferences: Help

You can set the following preferences for the help system:

Open help search

Here you can specify whether the help search will be opened in the Help View or in a webbrowser. If embedded web browsers are supported on your system an embedded help browser will be used instead of an external one.

Open help contents

Here you can specify whether the help contents will be opened in the Help View or in an external webbrowser.

Open dynamic help for dialogs

Here you can specify whether the dynamic help for dialogs will be opened in a dynamic help section or in an infopop.

Note: The settings on this page can affect the presentation of the Help View. If the selected browser is not fully compatible with Internet Explorer or Mozilla, or has JavaScript disabled, the Help View shown in the browser might be a simplified version.
