Preferences: Views

Here you can specify the following common preferences for several views:

Info

Here you can specify which additional information or hints shall be presented at the top of the graphical views. This includes information about the current selection as well as hints related to the edit/search mode.

Edit/Search Mode

Here you can specify whether or not the edit/search mode of the Box, Tab and Notes View can be entered via mouse double click or left via [Esc]-key, respectively.

Specific Preferences of certain Views

The following views provide additional specific preferences:

  • Chord Generation View
  • Chord Results View
  • Scale Results View
  • Scale Finder View
  • Fretboard View
  • Keyboard View
  • Box View
  • Tab View
  • Notes View
