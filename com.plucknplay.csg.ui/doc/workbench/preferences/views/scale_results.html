Preferences: Scale Results View

You can set the following preferences for the Scale Results View:

Entries per Page

Here you can define the number of result entries per page.
Note: The less entries you allow per page the more pages will be needed.

Visible Columns

Since this tabular view has many columns the view tends to become very wide and confusing. Here you can specify which columns shall be shown. You just have to select the columns you want to show. The first column, Scale, will always be shown and can not be hidden.
