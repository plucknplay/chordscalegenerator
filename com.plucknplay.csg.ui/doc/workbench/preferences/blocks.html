Preferences: Blocks

You can set the following preferences for the blocks that can be selected in the Fretboard View as well as highlighted in the Keyboard, Box, Tab and Notes View:

You can choose between fret blocks, advanced fret blocks and octave blocks in the View Menu. While octave blocks cover a whole octave, fret blocks cover all strings of your instrument in a defined fret span. For advanced fret blocks you can define aside a fret span an additional string span.

Fret Blocks

Here you can specify the fret span which should be used for fret blocks. Furthermore, you can define whether or not notes of empty strings (that belong to a selected chord or scale) shall be shown, too.

Fret Blocks

Here you can specify the fret and string span which should be used for advanced fret blocks. Furthermore, you can define whether or not notes of empty strings (that belong to a selected chord or scale) shall be shown, too.

Octave Blocks

Here you can set the preferences for the navigation between octave blocks in the Fretboard View. You can choose whether each note of a selected chord or scale can serve as starting note of an octave block or whether only root notes are allowed.
