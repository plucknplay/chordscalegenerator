File Menu

The File Menu of the main menu contains the following actions:

New → Instrument

Opens a blank Instrument Editor where you can define a new instrument. After you have saved the new instrument (see File → Save), it will be inserted into the main category of the Instruments View.

New → Chord

Opens a blank Chord Editor where you can define a new chord. After you have saved the new chord (see File → Save), it will be inserted into the main category of the Chords View.

New → Scale

Opens a blank Scale Editor where you can define a new scale. After you have saved the new scale (see File → Save), it will be inserted into the main category of the Scales View.

Save

Saves the element (instrument, chord or scale) in the currently active editor.

Save All

Saves all elements in currently opened editors.

Preferences ...

Opens the preferences dialog. The most recently selected page will be pre-selected.

Exit

Closes the application.
