View Menu

The View Menu of the main menu contains the following actions:

Notes Mode

Here you can specify the notes mode. Since certain notes can have several different names (e.g. C# corresponds to Db) you can specify which name(s) shall be used in the views of the application.

Root Note

Here you can specify the root note of a selected chord or scale in the Chords and Scales View. A selected chord or scale with the associated root note will be shown in the graphical views Fretboard, Keyboard, Box, Tab and Notes.

Previous/Next Root Note

These two menu entries (especially with the help of the associated key bindings) allow you to quickly change the root note.

Show Blocks

Here you can specify whether or not blocks (sections of chord schemas or scales) shall be shown. Blocks can be selected in the Fretboard View. Furthermore, they are highlighted in the Keyboard, Box, Tab and Notes View.

Fret Block Mode

Here you can activate the fret block mode. Fret blocks cover all strings of your instrument in a defined fret span.

Advanced Fret Block Mode

Here you can activate the advanced fret block mode. Advanced fret blocks span a defined number of frets and strings.

Octave Block Mode

Here you can activate the octave block mode. Octave blocks cover a whole octave starting from a selected note.

Change Hand

Here you can specify whether you are a right- or left-handed player.
