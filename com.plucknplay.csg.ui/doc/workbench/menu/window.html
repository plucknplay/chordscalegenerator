Window Menu

The Window Menu of the main menu contains the following actions:

New Editor

Creates a new instanceof the currently active editor.

Open Perspective

Opens the perspective you have selected in the submenu.

Show View

Opens the view you have selected in the submenu. If the view you want to open is not offered in the submenu, click Others... to open a dialog box which shows all available views.

Save Perspective

Saves the current arrangement of the editor area and all visible views in the currently active perspective.

Save Perspective As...

Saves the current arrangement of the editor area and all visible views in a chosen perspective. It is possible to overwrite existing perspectives or to create new ones.

Reset Perspective

Resets the currently active perspective.

Delete Perspectives...

Here you can choose the perspectives you want to delete through a dialog box.
Note: It is only possible to delete perspectives you have created yourself. The perspectives Setup, Chords and Scales can not be deleted.

Close Perspective

Closes the currently active perspective.

Close All Perspectives

Closes all opened perspectives.

Show Editor Area

Shows or hides the editor area.
Note: You can not hide the editor area when an editor is opened.

Navigation Submenu

The Navigation submenu of the Windows Menu contains the following actions:

Maximize Active View or Editor

Maximizes the currently active view or editor.

Minimize Active View or Editor

Maximizes the currently active view or editor.

Activate Editor

Activates the editor area.

Next Editor

Jumps to the next editor.

Previous Editor

Jumps to the previous editor.

Close

Closes all opened editors.

Close All

Closes all opened editors.

Next View

Jumps to the next view.

Previous View

Jumps to the previous view.

Next Perspective

Jumps to the next perspective.

Previous Perspective

Jumps to the previous perspective.
