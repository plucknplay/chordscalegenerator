Help Menu

The Help Menu of the main menu contains the following actions:

Welcome

Opens the Welcome Pages of the Chord Scale Generator. The Welcome Pages give you a first overview and a review of the main tasks of the application.

Help Contents

Opens the help contents in a browser. You can specifiy in the help preferences whether an internal or external browser shall be used.

Search

Opens the Help View to enter a search expression.

Dynamic Help

Shows useful help contents for a selected program element (view, editor or preference page). Here the corresponding help page of the selected element as well as other potential interesting help pages will be shown.

Video Tutorials

Dieser Menüeintrag leitet dich direkt auf die Video-Tutorial-Seite der Projekt-Website.

Support

What's New?

Search for Updates

Checks whether or not new updates are available.

About

Opens the About Dialog.
