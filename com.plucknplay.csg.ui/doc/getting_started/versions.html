Versions

You are allowed to use the test version of the Chord Scale Generator as long as you want. This gives you enough time to properly test the Chord Scale Generator before you consider to buy the full version.

The test version offers almost the whole range of operations. You are allowed to perform all calculations on any instrument and it is also possible to export the results in order to create your own chord, scale or song books.

However, the number of the predefined chords and scales is, of course, limited in the test version. There is only a limited number of chords and scales available and it is not possible to edit them or even create new chords or scales. Therefore, you can only perform the calculations on a certain number of chords and scales.

Note: With these restrictions, you may sometimes find the results unsatisfactory, especially the reverse calculations in order to find corresponding chords or scales for a given fingering pattern or musical notation This is due to the fact that the program can only work with chords and scales which are already defined.

Nevertheless, we hope you are able to properly test the Chord Scale Generator and you will get a first insight into the range of useful operations that this program offers you. If you decide to purchase this software you will need an activation key, which you can buy on our website.

Note: In order to activate the Chord Scale Generator on the computer you want to use it on, you need to have internet access for a short time.
