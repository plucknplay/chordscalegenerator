Features

Supports All Stringed Instruments

  • The Chord Scale Generator supports all instruments with 2 to 12 strings or doubled strings. This includes instruments such as guitar, bass, banjo, mandolin, violin, viola, cello, balalaika, bouzouki, cavaquinho, charango, domra, and many more.
  • The settings for many instruments with their most common tunings are already be predefined in the Chord Scale Generator.
  • It is possible to create new instruments. You can specify the name, dimensions (string and fret number) and tuning of the instrument, as well as more advanced settings like the sound, scale length and positions of the capo (tasto).

Supports All Chords

  • The Chord Scale Generator supports all possible chords.
  • A multitude of the most common chords are already predefined, like for instance maj, min, maj7, aug, dim, m7, add9, add11 and many more.
  • It is possible to create new chords.
  • The Chord Scale Generator also supports different notations for a chord.

Supports All Scales

  • The Chord Scale Generator supports not only chords but also all possible scales.
  • The most common scales, like the ecclesiastical modes and some pentatonics, are predefined.
  • It is possible to create new scales.

Determines All Fingerings for a Certain Chord

  • The Chord Scale Generator helps you to determine all possible fingerings for a certain chord.
  • The Chord Scale Generator works with a powerful algorithm, rather than rigid database structures, to generate the fingerings.
  • In order to reduce the size of the result set a wide range of restrictions are available. Possible restrictions include the level, the grip range, and the bass or lead tone.

Determines chords to fit a fingering pattern or musical notation

  • The Chord Scale Generator offers a reverse calculation to find corresponding chords to fit a given fingering pattern or musical notation.

Determines scales to fit a melody or riff

  • The Chord Scale Generator helps you to find corresponding scales to fit a given melody or riff.
  • This function finds all scales which contain at least the notes of your given melody. The additional notes of a found scale can easily be used to extend your melody.

Displays Chords and Scales in Different Graphical Views

  • The Chord Scale Generator displays the calculated chords and scales in different graphical views.
  • Chords and scales can be shown on a stylised fretboard, a keyboard, in tablature, in a box view or even in musical notation.

Sound Support

  • Play any chord or scale to hear it sounds.
  • The played tones are graphically highlighted while they are being played.
  • You can configure how the played chords and scales sound.

Export of Chords and Scales

  • It is possible to export all graphical represenations of chords and scales.
  • Use the Chord Scale Generator to create your own chord, scale and song books.

Different Representations for Left- and Right-Hander

  • The Chord Scale Generator provides settings for left and right hander. You can also define if you play the instrument as a lefty or righty.
  • Depending on a left or right hander setting, all graphical representations will be adjusted.

Customizable User Interface

  • The user interface of the Chord Scale Generator consists of a multitude of views and editors.
  • The views and editors can be freely arranged to perspectives. Therefore, it is possible to define a separate perspective for each task you want to perform with the Chord Scale Generator.
  • Besides, many views can be adjusted to your own preferences.
  • Thus, it is possible to decide completely how the Chord Scale Generator shall look like for its specific tasks.

Update Support

  • The Chord Scale Generator provides an update action which keeps the program up-to-date.

Detailed Help Available

  • The Chord Scale Generator provides context-specific help which gives detailed information about each program element.
