Update Support

In the last chapter of this tour we finally want to explain the update support of the Chord Scale Generator. The Chord Scale Generator provides an update action which enables you to keep your program up-to-date. Therefore, it is not necessary to install the program again whenever a new version is available.

Updates are important to improve the program. They can either provide new features or fix little bugs (errors). Thus, it is advisable to check for new updates frequently. But don't worry, an update never changes your current program settings or data (instruments, chords or scales).

You can start the search for an update with the menu entry Help → Search for Updates. In order to search for new updates an active internet connection is necessary. If a new version is found a corresponding dialog box opens which informs you about the latest available version. After accepting the lincence agreement the new version will be downloaded and installed. It is necessary to restart the Chord Scale Generator so that the changes can take effect.

Note: The download of the new version can take some time depending on the speed of your internet connection. However, the update process can be performed in the background so that it is possible to work with the Chord Scale Generator in the meantime.

There is also the possibility of an automatical search for updates. In the update preferences (File → Preferences... → Update) you can define in detail when and how such an automatic update shall be performed.
