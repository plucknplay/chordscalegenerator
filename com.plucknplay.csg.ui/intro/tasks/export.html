Export Actions

In this chapter we want to throw light on the export functions of the Chord Scale Generator. The content of all graphical views (which are the views Box, Tab, Notes, Fretboard and Keyboard) can be exported as image file or copied into the clipboard. In order to export or copy the current content you just have to perform the corresponding popup menu entry of the respective view.

By pasting the contents of the clipboard or including exported image files into other programs (e.g. Office applications, ...) you can easily arrange your own chord or song books. Since the arrangement of chord books by copying every single interesting chord would probably turn out to be very tiring, a multiple clipboard export is provided. Starting from the Chord Results View it is possible to copy several chords simultaneously into the clipboard. To do so you have to mark the chords of interest in the first table column. The popup menu entry Copy to Clipboard performs the clipboard export for the marked chords. The contents of the clipboard can be pasted into other programs afterwards.

It is also easily possible to create your own scale books. The popup menu entry Copy Scale Blocks to Clipboard of the views Scales and Scale Results copies image data of all scale blocks of a selected scale into the clipboard. It depends on which scale block mode you have chosen in the scale block preferences which scale blocks of a selected scale will be exported (File → Preferences... → Scale Blocks).

You can define in the export preferences which image data shall be copied for an individual chord or scale block as well as the size of the exported image data (File → Preferences... → Export)
